Ways of adding styles

  • Inline Approach
  • Internal Approach
  • External Approach
Inline CSS

it is use "style" attribute

Hey it's me Sam

I am a Coder

I studied in University of Kalyani

This approach is only used to apply styles for a specific tag only, if we wants to apply the styles we have to repeat the same styles all the required HTML Tags. This approach increases the code duplication and code redundancy. This increase code maintainability of the application

Internal CSS

It is use < style > tag in the head section of HTML

Hello World

Hey world

Hola World

But in this approach we can apply the styles only for the specific HTML Document, these styles will not be applied to any other HTML Document

External CSS

This is the best approach to apply styles to any webpage. In this approach we will create a separate ‘.css’ file and apply all the styles to it. We can connect this ‘.css’ file to any HTML document using a HTML < link > Tag.
