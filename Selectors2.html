Attribute Selectors

By using CSS we can even select a specific attribute of a HTML Element and apply styles to it. These styles are used when we apply some styles only for a text box or password box. As both are using an < input /> tag, we can differentiate them using an attribute only.



Pseudo Selectors

An action which a user is performing on the webpage is called as an event. This is the selector to apply the CSS styles for some special events on the browser.
