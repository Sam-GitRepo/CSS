CSS Position

The CSS Positions are used to place the HTML Elements in the specified positions on the webpage. For these styles we use a CSS property called position.

There are 5 position

static: default position of any html element

relative: this positioned relative to its own

absolute: this is positioned relative to its parent

fixed: this is fix an element in the given position

sticky: this is to stick any element in the given position
