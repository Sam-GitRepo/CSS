CSS Colors

    There are 6 types of Colors

  • Textual color : red, blue
  • RGB Eg: rbg(red, green, blue) range : 0 - 255 white : rgb(255,255,255); black : rgb(0,0,0);
  • HEXA Eg: #rrggbb Range: 00-FF; white:#FFFFFFF; black:#000000;
  • HSL(hue,saturation,lightness); hue: degree on color wheel -> 0 - 360 saturation : darkness % -> 0- 100% lightness : lightness % -> 0 - 100% eg: hsl(60,80%,40%);
  • RGBA -> same as RGB & A for alpha alpha 0 -> full transparent, 1 -> full color syntax :rgba(0,0,0,1);
  • HSLA(Hue-saturation-lightness-alpha) syntax: hsla(60,80%,90%,0.5);
